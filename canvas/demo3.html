Importing Images

Importing images into a canvas is a two-step process:.

  1. Get a reference to an image (e.g., HTMLImageElement)
  2. Draw the image to the canvas using drawImage()

Important: the image needs to be loaded before drawing to the canvas (otherwise the draw may fail silently or the browser might issue an exception). Ensure this by either wrapping the draw command in an event listener ('load') or by preloading images.

Get reference to Image

Images on the same page

Get a reference to the images on the page using one of the following:

  • document.images
  • document.querySelectorAll('image')
  • document.querySelector('#{id}')

Scaling Images

A second variant of the drawImage method enables scaling of an image.

The syntax is as follows:

drawImage(image, x, y, width, height)

Slicing (Cropping) Images

A third variant of the drawImage method enables cropping of an image.

The syntax is as follows:

drawImage(image, sx, sy, sWidth, sHeight, dx, dy, dWidth, dHeight)

Where sx, sy, sWidth, sHeight describe the cropped portion of the source image, and dx, dy, dWidth, dHeight describe the position and sizing in the destination (the canvas)
