Importing Images

Importing images into a canvas is a two-step process:.

  1. Get a reference to an image (e.g., HTMLImageElement)
  2. Draw the image to the canvas using drawImage()

Get reference to Image

Images on the same page

Get a reference to the images on the page using one of the following:

  • document.images
  • document.querySelectorAll('image')
  • document.querySelector('#{id}')
