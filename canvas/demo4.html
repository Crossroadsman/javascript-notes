Canvas State

Canvas state is stored on a stack. Every time the canvas is saved, the current stage is pushed onto the stack. Each restore pops the most recent saved state from the stack.

The state comprises:

  • Any transformations that have been applied;
  • The current value of various atttributes
  • The current clipping path

Rotation

We can use rotation just like we use translate.

We can also combine rotation and translation, for example, placing an element, then translating to the center of the element to perform the rotation (so that the element appears to rotate in place) then translate back to 0,0.
