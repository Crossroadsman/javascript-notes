Rect

Unlike SVG, canvas has only one primitive polygon: rectangle.

We can draw using fillRect, strokeRect and clearRect.

rects draw immediately to the context (unlike paths)

Path

A path is a list of points, connected by segments of lines.

The lines can be joined with straight or curved lines.

Each segment can be styled differently.

A path, or subpath, can be closed.

Process

  1. Begin a path (beginPath())
  2. Move to the initial position using moveTo
  3. Use drawing commands (e.g., lineTo()to draw into the path
  4. Issue closePath() to add a straight line to close the path
  5. Use stroke or fill to render the path
